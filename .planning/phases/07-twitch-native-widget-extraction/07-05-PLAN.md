  <how-to-verify>
**Load the extension:**
1. Open `chrome://extensions`, enable Developer mode
2. Click "Load unpacked" and select the `dist/` directory
3. (Or reload if already loaded)

**Test 1: Tab bar on Twitch (D-04, D-05, D-06)**
1. Navigate to a live Twitch stream (e.g., twitch.tv/etro or any AllChat-enabled streamer)
2. Verify: Tab bar appears at top of chat area with two tabs: [Infinity AllChat] | [Twitch Chat]
3. Verify: Tab bar uses dark theme styling (not Twitch purple)
4. Verify: Connection dot visible in AllChat tab label
5. Verify: InfinityLogo (infinity symbol) visible before "AllChat" text

**Test 2: Tab switching (D-07)**
1. Click "Twitch Chat" tab
2. Verify: AllChat iframe disappears, native Twitch chat is fully visible
3. Verify: Tab bar stays at top (does NOT disappear)
4. Verify: "Twitch Chat" tab shows active indicator (purple bottom border)
5. Click "AllChat" tab
6. Verify: AllChat iframe reappears, native Twitch chat is hidden again
7. Verify: Chat messages are still present (no data loss on switch)
8. Verify: No old "Switch to native" button visible anywhere

**Test 3: Channel points (D-15)**
1. While on AllChat tab, check bottom of chat area
2. Verify: Channel points balance is visible (if logged into Twitch)
3. If a "claim bonus" button appears, try clicking it
4. Verify: Points are actually claimed (balance updates)

**Test 4: Predictions/Polls (D-14, D-16)**
(Only testable if the streamer runs a prediction or poll — skip if not available)
1. When a prediction appears, verify it shows in the top widget zone
2. Try voting on the prediction clone
3. Verify: Vote registers on the real prediction

**Test 5: Pop-out mode**
1. Click the pop-out button (small arrow icon, top-right of iframe)
2. Verify: Pop-out window opens with full header (NOT tabBarMode)
3. Close pop-out, verify in-page chat returns normally

**Test 6: YouTube regression check**
1. Navigate to a live YouTube stream with AllChat enabled
2. Verify: Normal iframe header appears (InfinityLogo center, connection dot, etc.)
3. Verify: "Switch to native" button works as before
4. Verify: No tab bar appears on YouTube (tab bar is Twitch-only)

**Test 7: Kick regression check**
1. Navigate to a live Kick stream with AllChat enabled
2. Verify: Normal iframe header, switch to native works, no tab bar
