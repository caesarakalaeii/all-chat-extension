**KickDetector class:**

`platform = 'kick' as const`

`isLiveStream()`:
- Primary: `document.querySelector('[data-state="live"]')` — returns true if found (update selector to checkpoint-confirmed value)
- Fallback: find first visible element from `['#channel-chatroom', '#chatroom', '.chatroom-wrapper']`; check `offsetParent !== null`; return true if found and visible
- If neither: `console.warn('[AllChat Kick] No live signal found, not injecting'); return false`

`async init()`:
- Check `if (!this.isLiveStream()) return;`
- Then `return super.init();`

`extractStreamerUsername()`:
- `window.location.pathname.match(/^\/([^/]+)/)`
- Exclude: `['home', 'categories', 'search', 'subscriptions', 'settings', 'clip', 'clips']`
- Return match[1] or null

`getChatContainerSelector()`:
- Return `['#channel-chatroom', '#chatroom', '.chatroom-wrapper']`

`hideNativeChat()`:
- Idempotent style-tag pattern with id `allchat-hide-native-style`
- CSS rule: `#channel-chatroom { display: none !important; }` (update to confirmed primary selector)
- Add date-comment: `/* verified: 2026-03-12 — update if selector breaks */`

`showNativeChat()`:
- Remove `#allchat-hide-native-style` style element

`removeAllChatUI()`:
- Remove `#allchat-container` element

`createInjectionPoint()`:
- Try selectors in sequence with date-comments:
  ```typescript
  const SELECTORS = [
    '#channel-chatroom',   // primary — verified 2026-03-12
    '#chatroom',           // fallback 1 — verified 2026-03-12
    '.chatroom-wrapper',   // fallback 2 — verified 2026-03-12
  ];
  ```
- Loop: for each selector, `try { slot = await this.waitForElement(sel); break; } catch { /* try next */ }`
- If no slot found: `console.warn('[AllChat Kick] No chat slot found — native chat remains visible'); return null;`
- Create container div with `id="allchat-container"` and `style="width: 100%; height: 100%;"`
- Append container inside slot (Twitch append pattern — slot.appendChild(container))
- Return container

**Module-level code (mirrors youtube.ts):**

```typescript
let globalDetector: KickDetector | null = null;
```

`initialize()` async function:
- Log content script loaded
- `getSyncStorage()` check — if disabled, return
- `globalDetector = new KickDetector()`
- Call `setupGlobalMessageRelay()` immediately
- Call `globalDetector.init()` (no waitForElement pre-check needed — isLiveStream() handles this)
- Call `setupUrlWatcher()`

`handleExtensionStateChange(enabled: boolean)`:
- If disabled: teardown globalDetector, set to null

`setupGlobalMessageRelay()`:
- Mirror youtube.ts exactly — but use `iframe[data-platform="kick"]` selector for relay targets
- Use extensionOrigin (NOT '*') for ALL postMessage calls in this function
- Handle EXTENSION_STATE_CHANGED, CONNECTION_STATE/WS_MESSAGE relay, GET_CONNECTION_STATE response

`setupUrlWatcher()`:
- `let activeUrl = location.href;`
- `handleNavigation()`: URL equality dedup check, teardown, isLiveStream() re-check, init
- `window.addEventListener('popstate', handleNavigation);`
- MutationObserver on `document.querySelector('title') || document.head` with `{ childList: true, characterData: true, subtree: true }`
- (NOTE: CONTEXT.md specified "pushState intercept (monkey-patch approach)" as the locked mechanism, but this is technically infeasible from an isolated content script world — monkey-patching only affects the content script's sandboxed scope, not the page. MutationObserver on title achieves the same SPA navigation detection goal. See RESEARCH.md Pattern 5 Anti-Patterns. Do NOT monkey-patch history.pushState.)

End file with `initialize();`
