<objective>
Replace YouTube's inline-style hide and MutationObserver URL polling with slot injection using a `<style>` tag hide and `yt-navigate-finish`/`popstate` event listeners.

Purpose: Inline `element.style.display = 'none'` is erased when Polymer recreates `ytd-live-chat-frame` on navigation. The `<style>` tag survives re-renders. The MutationObserver URL watcher fires thousands of times per navigation; `yt-navigate-finish` fires once and is the canonical YouTube SPA signal.

Output: `src/content-scripts/youtube.ts` rewritten with slot injection, style-tag hide, and event-based navigation.
</objective>

    function setupUrlWatcher(): void {
      let activeUrl = location.href;
