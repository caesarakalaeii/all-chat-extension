  <name>Task 1: Upgrade PlatformDetector — waitForElement, teardown, async signature</name>
  <files>src/content-scripts/base/PlatformDetector.ts</files>
  <action>
    Make three changes to PlatformDetector.ts:

    1. Add `protected waitForElement()` method implementing Promise-based polling:
       - Parameters: `selector: string`, `timeoutMs = 10_000`, `preDelayMs = 200`, `pollIntervalMs = 100`
       - Returns: `Promise<HTMLElement>`
       - Implementation: after preDelayMs delay, check once immediately; if not found, poll every pollIntervalMs until timeoutMs elapses; resolve with element on success; reject with `new Error('[AllChat] waitForElement: "${selector}" not found after ${timeoutMs}ms')` on timeout
       - The caller (Twitch/YouTube createInjectionPoint) is responsible for catching rejection

    2. Add `teardown(): void` method (non-abstract, overrideable):
       - Remove `#allchat-container` from DOM if it exists
       - Remove `#allchat-hide-native-style` from DOM if it exists
       - Call `this.showNativeChat()`
       - Log: `[AllChat ${this.platform}] Teardown complete`
       - Subclasses may call `super.teardown()` and add extra steps

    3. Update `createInjectionPoint()` abstract signature from sync to async:
       - Change: `abstract createInjectionPoint(): HTMLElement | null;`
       - To:     `abstract createInjectionPoint(): Promise<HTMLElement | null>;`
       - In `init()` at the call site: change `const container = this.createInjectionPoint();` to `const container = await this.createInjectionPoint();`
       - The `init()` method is already `async` — no further changes needed there

    TypeScript strictness note: `strict: true` is on. All null checks must be explicit. `document.querySelector()` returns `Element | null` — cast to `HTMLElement | null` after null check.

    Do NOT change any other existing methods. The `findChatContainer()` method, `injectAllChatUI()`, `checkStreamerExists()`, and all abstract declarations other than `createInjectionPoint()` remain untouched.

        const order = await page.evaluate(() => {
          const container = document.getElementById('allchat-container');
          const native = document.querySelector('ytd-live-chat-frame');
          if (!container || !native) return null;
          const pos = container.compareDocumentPosition(native);
          return (pos & Node.DOCUMENT_POSITION_FOLLOWING) !== 0;

        expect(order).toBe(true);
      });
    });

        await expect(inlineHidden).not.toBeAttached();
      });
